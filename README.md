📝 Blog Web Application

A full-stack blog application built using Node.js, Express, MongoDB, and EJS. Users can create, edit, view, and delete blog posts with a clean and structured interface.

⸻

🚀 Features
	•	✍️ Create new blog posts
	•	📖 View all posts
	•	🔍 View individual post
	•	✏️ Edit existing posts
	•	❌ Delete posts
	•	🗂️ MVC architecture
	•	🎨 Server-side rendering using EJS

⸻

🛠️ Tech Stack
	•	Backend: Node.js, Express.js
	•	Database: MongoDB
	•	Frontend: EJS, CSS
	•	Tools: Git, GitHub

⸻
